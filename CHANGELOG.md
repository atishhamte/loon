# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial implementation of LOON format encoder and decoder
- TypeScript support with full type definitions
- Comprehensive test suite with unit and property-based tests
- CommonJS and ESM module support
- Documentation and examples

## [1.0.0] - TBD

### Added
- First stable release
- `encode()` function for converting JavaScript values to LOON format
- `decode()` function for parsing LOON format strings
- Support for all JSON data types
- Error handling with descriptive messages
- Zero runtime dependencies
